## Definition and Scope
An “AI Scientist” system is an AI-enabled workflow that supports scientific discovery end-to-end: finding and synthesising knowledge, generating hypotheses, designing tests, executing or assisting experiments, analysing results, and communicating conclusions with traceable evidence.

This list focuses on practical resources for building, evaluating, and governing such systems.

## Core Capabilities
- Literature intelligence (search, retrieval, synthesis, citation hygiene).
- Scientific planning (hypothesis generation, experiment design, protocols).
- Tool-use (code execution, simulators, data pipelines, lab or workflow tools).
- Evaluation (benchmarks, reproducibility, error analysis).
- Communication (writing, figures, argument structure).
- Safety (dual-use awareness, governance, oversight).

## Reference Architecture
A typical AI Scientist system can be viewed as:
1) Knowledge layer → 2) Planning layer → 3) Tool-use/execution layer → 4) Evaluation layer → 5) Reporting layer → 6) Governance layer.
